Space Journey

Summary

Space Journey was a group project that was created in p5.js for a engineering class. The main focus of the project was on making the environment and process similar to working with a professional team on a budget. We had a deadline, sprint organization, peer evaluations, and a presentation for the stakeholders. Even though this project was for a class my team and I thoroughly enjoyed working together and took it very seriously. The game's target audience was suppose to be for children so we kept the games we made fairly easy while also making them fun and engaging. The game is set in space with each activity keeping true to the theme.

Gameplay

There are three accessable games through the menu. There's a maze game where the player needs to draw their cursor to the end of the level without touching any walls. If a wall is touched a "You Lose" text will prompt on the screen but the player only has to go back to start to restart. After the player is done with the game they may click on the "menu" button on the top left of the screen to go back to the menu and select another game. The second game is an aim trainer. The player needs to click on the targets as they appear but quickly before the timer runs out. When the timer runs out the game will end and the score they get will be thier final score. After the player may click on the Menu button to choose another game. The final game is a memory matcher. The player must memorize the colors as they appear on the prompt as they will need to enter those colors in a sequence at the end of the round. If the player gets the sequence correct then they gain a score of +100 but if they get it wrong then they will lose a life. Once all lives are lost the game ends and whatever the player's score is before then is their final score. The player may then click on the "menu" button to select another game.

Implemented Features
