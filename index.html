Rogue Nebula (2023)

A project made for the Video Game Design Club during their summer session. The game was meant to be a prototype for Unstable Star. Endlessly plow through waves of enemies in the pursuite of a high score. The game is heavily inspired by classic arcade games like Galaga, Defender 1981, and Space Invaders. It features a scrolling 3D environment, a reactive user interface, vibrant special effects, and an assortment of enemy behaviors.

Future Scape (2024)

A project I was tasked to develop for Enterprise Technologies. This project was to be a virtual career explorer targeted to students in primary and middle school. Featuring multiple minigames that explored different careers like a pharmicology, manufacturing, forensic science, etc. I developed the pharmacy, manufacturing, and urban planning minigames. As a pharmacist give your customers the medication they need in a certain amount of time. In manufacturing build assebly lines and combine components to make products that you can sell. In urban planning build your own city while managing a budget and resources at your disposal.

Unstable Star (2024)

A project made for the Video Game Design Club during their 2023-24 session. Enjoy and immerse yourself in a comedic and heartwarming story about a lowly guard taking on an empire. Travel through three variable levels including dynamic objectives and challenging boss fights. This game features an animated dialogue system, weapon and ability inventory and selection system, a reactive user interface, vibrant sounds and vfx, and an assortment of enemy behaviors.

The Devil's Diner (2023)

A project made with a small team for the Ludum Dare Game Jam #50, which is a challenge where competitors have 72 hours to create a game completely from scratch. Our game stars a doomed chef who must cook for demons in Hell's most treacherous kitchen. This game features a crafting system, interactable UI, a leaderboard system, cutscenes, and a level system. It was a blast working with each other on this project, so much so that we decided to port the game to mobile, which is currently in progress. All assets and artwork are my own.

A project made with a friend for the Ludum Dare Game Jam #49. This game features sophisticated AI, pathfinding using A*, a dynamic NPC scheduling system, a day and night cycle, and an inventory system. All assets and artwork are my own.

Star Raid (2021)

Star Raid was a very ambitious project that I was developing with one of my colleagues. Unfortunately, development for the game ceased due to the increasingly large scope of the project and our desire to move forward with more interesting projects overweighed our decision. This game was centered around co-op gameplay and featured P2P client-based multiplayer, a crafting system, A* pathfinding for enemies, raycasting projectiles, and a user-based command system. All assets and artwork are my own.

Fishy Business (2023)

Fishy Business was a group project created for a game developement class using the Gamemaker engine. The main focus of the project was to simulate an environment and process similar to working with a professional studio. We had deadlines, checkups, peer evaluations, design documentation, and presentations. I was the group leader and the main gameplay programer. A snapshot of this project can be found on our itch.io page.
