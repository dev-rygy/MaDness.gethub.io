Bloodlust

A project made for the Ludum Dare Game Jam #49 which is a challenge where competitors have 72 hours to create a game completely from scratch. This game features dynamic AI and pathfinding using a pathfinding algorithm called A*. Also a dynamic NPC scheduling system, day and night cycle, and a complete inventory system are just some of the amazing features we were able to implement in the 72 hour deadline. All assets and artwork are my own.
